# Web-Based Job Portal Application

## 📌 Project Overview

The Web-Based Job Portal Application is a full-stack web application designed to connect job seekers and recruiters on a single platform. The system allows recruiters to post job openings and manage applicants, while job seekers can search and apply for jobs easily.

This project was developed to improve skills in Java Full Stack Development, including backend development, REST APIs, database management, and frontend integration.

---

## 🚀 Features

### 👨‍💼 Job Seeker Features

* User Registration & Login
* Search and Filter Jobs
* Apply for Jobs
* Resume Upload
* View Applied Jobs

### 🏢 Recruiter Features

* Recruiter Registration & Login
* Post New Jobs
* Manage Job Listings
* View Applicants
* Update/Delete Job Posts

### 🔐 Authentication & Security

* Role-Based Access Control
* Secure Login System
* Session Management
